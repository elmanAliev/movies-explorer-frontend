## Запуск проекта

1. Клонировать репозиторий

    `git@github.com:elmanAliev/movies-explorer-frontend.git`

2. Установить зависимости

    `cd movies-explorer-frontend`
    
    `npm install`

3. Для запуска использовать команды

    `npm start` — запускает сервер
